# Meta-DB Visualizer

**Універсальний графічний інтерфейс управління базою даних PostgreSQL.**

Додаток працює за принципом "Metadata-First" — він автоматично підключається до будь-якої вашої бази даних PostgreSQL, аналізує її структуру та будує зручний візуальний граф для управління таблицями та зв'язками.

## 🚀 Як запустити проєкт

Для роботи потрібен встановлений [Node.js](https://nodejs.org/).

1. Завантажте проєкт та відкрийте папку в терміналі.
2. Встановіть усі необхідні залежності:
   ```bash
   npm run install:all
   ```
3. Запустіть проєкт:
   ```bash
   npm run dev
   ```
4. Відкрийте браузер за посиланням [http://localhost:5173](http://localhost:5173) та введіть дані доступу до вашої PostgreSQL бази даних.
