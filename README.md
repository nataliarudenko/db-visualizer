# Meta-DB Visualizer

Додаток для візуалізації та управління структурою бази даних PostgreSQL за принципом "Metadata-First".

## 🚀 Як запустити проект

1.  **Попередні вимоги**:
    Переконайтеся, що у вас встановлено [Node.js](https://nodejs.org/) (версія 18 або новіша).

2.  **Встановлення залежностей**:
    Відкрийте термінал у кореневій папці проекту та виконайте:
    ```bash
    npm install
    ```
    *Ця команда встановить бібліотеки одночасно для фронтенду та бекенду.*

3.  **Запуск у режимі розробки**:
    ```bash
    npm run dev
    ```

4.  **Використання**:
    *   Відкрийте браузер за адресою: **http://localhost:5173**
    *   Введіть дані для підключення до вашої бази даних PostgreSQL.

---

### Порти за замовчуванням:
*   **Бекенд**: http://localhost:3001
*   **Фронтенд**: http://localhost:5173
